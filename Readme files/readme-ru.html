ВВЕДЕНИЕ
PHP-Fusion является системой управления контентом (CMS), с открытым исходным кодом, написанная в PHP 5. Она использует MySQL базу данных для хранения содержимого Вашего сайта и включает в себя простую и удобную систему администрирования. PHP-Fusion включает в себя стандартные функции, подобных CMS.

УСТАНОВКА
Перед установкой PHP-Fusion Вам необходимо проверить требования к данной версии:

- PHP 5.1.2 или выше
- MySQL 4.1 или выше

Вы должны создать базу данных MySQL. Вы можете сделать это через Вашу панель управления хостингом или PhpMyAdmin. Убедитесь, что у Вас есть все данные для MySQL, включая имя хоста, имя пользователя, пароль и имя базы данных. Вам необходимо указать эти данные во время установки.

1. Перед установкой переименуйте файл _config.php (распроложен в папке files) в config.php.

2. Скопируйте файлы пакета на Ваш веб-сервер.

3. В большинстве случаев Вам понадобится установить права доступа CHMOD, на следующие файлы и папки в 777:

  • administration/db_backups/
  • downloads/
  • downloads/images/
  • downloads/submissions/
  • downloads/submissions/images/
  • forum/attachments/
  • ftp_upload/
  • images/
  • images/imagelist.js
  • images/articles/
  • images/avatars/
  • images/news/
  • images/news/thumbs/
  • images/news_cats/
  • images/photoalbum/
  • images/photoalbum/submissions/
  • config.php
Примечание: Некоторые хосты не позволяют устанавливать CHMOD 777, в этом случае Вы можете использовать CHMOD 755, если не удается выставить CHMOD 777.

4. Перейдите на Ваш сайт, где setup.php запустится автоматически. Если нет, то запустите setup.php вручную, введя свой полный адрес сайта /setup.php. Пример: http://www.yourdomain.com/setup.php.

5. Выполните полный процесс установки, следуя всем инструкциям на экране.

6. Сразу после завершения установки PHP-Fusion вы должны установить CHMOD config.php обратно в 644 и удалить setup.php с веб-сервера. Невыполнение этого требования может привести к проблемам с доступом к Вашему сайту.

ОБНОВЛЕНИЕ С ВЕРСИЙ v7.00.XX ИЛИ v7.01.XX
Перед обновлением настоятельно рекомендуется сделать резервные копии Ваших файлов и базы данных, так как PHP-Fusion 7,02 является одним из основных обновлением с PHP-Fusion 7,01. Вы должны точно следовать этим инструкциям. Мы также рекомендуем Вам сделать обновление 7.00.xx к 7.01.xx, прежде чем Вы обновитесь до 7.02.00.

Перед обновлением необходимо проверить требования для этой версии:

- PHP 5.1.2 или выше
- MySQL 4.1 или выше

1. Код версия 7,02 является наиболее совместимым с версией 7.01, но поскольку есть много изменений в ядре, некоторые аддоны (моды, плагины, панели или темы) могут не работать должным образом.

2. Первым делом необходимо загрузить файл обновления из папки 'upgrade v7' (или 'upgrade v701', если у Вас уже используется v7.01) в папку administration на Вашем хостинге, а также загрузить папку с локалями, так как без них некоторые локали не будут добавлены в базу данных.

3. Войдите на сайт под Супер Администратором. В разделе - Администрирование системы, нажмите на кнопку Обновление. Вы должны следовать пунктам обновления, процесс завершится только тогда, когда появляется надпись "Обновление базы данных завершено".

4. ОЧЕНЬ ВАЖНО: Поскольку этот релиз содержит ряд структурных изменения, некоторые элементы сайта, не будут работать должным образом, пока Вы не обновите все файлы ядра. Вы должны обновить ВСЕ файлы из пакета релиза, КРОМЕ:

  • _config.php
  • setup.php

Установите CHMOD 777 на следующие файлы и папки:

  • downloads/
  • downloads/images/
  • downloads/submissions/
  • downloads/submissions/images/
  • ftp_upload/

Если у Вас есть какие-либо сомнения, пожалуйста, не стесняйтесь обратиться за помощью на одном из наших сайтов поддержки, есть много опытных пользователей в нашем сообществе, которые могут помочь или дать Вам советы относительно процесса обновления.

МЕЛКИЕ ИСПРАВЛЕНИЯ НА НЕПРОФИЛЬНЫЕ ТЕМЫ
Если вы используете непрофильные темы для Вашего сайта, Вам необходимо отредактировать файл theme.php для того, чтобы Ваша тема отображала изображения категории новости.

Изображение категории новости:

  • Откройте файл themes/ваша_тема/theme.php
  • Найдите функцию render_news
  • Найдите вот это ".$news."
  • Замените на это ".$info['cat_image'].$news."

Аватары в комментариях:

  • Откройте файл themes/ваша_тема/theme.php
  • Найдите функцию render_comments
  • Добавьте вот это if ($settings['comments_avatar'] == "1") { echo "".$data['user_avatar']."\n"; }
  • Не забудьте добавить в styles.css класс .comments_avatar { } и настроить его, как вам нравится.
Сохранить и загрузите файл.
СОВЕТЫ ПО БЕЗОПАСНОСТИ
Вот несколько полезных советов, которые помогут сохранить Ваш сайт безопасности:
  • Убедитесь, что файл config.php не доступен для записи (должно быть CHMOD в 644).
  • Всегда удаляйте файл установки setup.php с Вашего хостинга после установки PHP-Fusion.
  • Назначайте разные пароли для FTP и MySQL.
  • Никогда не позволяйте таких вложений на форуме, как: php, html, exe, или любой тип текстового файла.
  • Используйте только аддоны, которые были протестированы и одобрены как безопасные.
